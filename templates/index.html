API Books Tech Challenge

Bem-vindo(a)! Acesse os endpoints da API abaixo:

Endpoints

  • /api/v1/books - Lista todos os livros
  • /api/v1/books/<id> - Detalhes de um livro
  • /api/v1/books/search - Buscar livros por título e/ou categoria
  • /api/v1/categories - Lista todas as categorias
  • /api/v1/health - Health check da API

Sobre

Esta API é parte do Tech Challenge e fornece dados de livros extraídos do site Books to Scrape. Pode ser utilizada para aplicações de ciência de dados, recomendações de livros ou consumo geral de dados.

Atualização de dados

Para atualizar os dados, execute o script de scraping:

python scripts/scrape_books.py

Depois, reinicie a API ou faça push para o Heroku.

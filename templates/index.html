API Books Tech Challenge

Access and explore API endpoints for data, insights, and ML-ready features.

Core Endpoints

/api/v1/books

List all books

/api/v1/books/<id>

Get details of a specific book

/api/v1/books/search

Search books by title and/or category

/api/v1/categories

List all categories

/api/v1/health

Check API health

Insights Endpoints

/api/v1/stats/overview

General statistics of the collection

/api/v1/stats/categories

Statistics by category

/api/v1/books/top-rated

Top 10 rated books

/api/v1/books/price-range

Filter books by price range

ML-Ready Endpoints

/api/v1/ml/features

Features prepared for ML models

/api/v1/ml/training-data

Training dataset with features and target

/api/v1/ml/predictions

Receive features and return prediction (POST)

Data Update

Run the scraping script to update the dataset:

python scripts/scrape_books.py

Then restart the API or push to Render.
